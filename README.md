SQL Sales Analytics and Power BI Dashboard
This project combines SQL-based analytical querying with business‑intelligence dashboard design. The goal is to transform raw sales data into meaningful insights for managers and decision‑makers.

1. Project Motivation
Organizations rely heavily on SQL and BI tools to analyze revenue patterns, monitor KPIs, and guide business strategy. This project demonstrates applied data‑analysis and reporting skills, essential for coursework and real‑world projects in the ACIT program at OsloMet.

2. Project Overview
The dataset includes:

Product categories
Regional sales
Unit price and quantity
Daily revenue records
3. SQL Analysis
Included SQL queries deliver:

Revenue by region
Top‑selling products
Daily revenue trends
KPI aggregations
Ranking and grouping analysis
4. Power BI Dashboard
The dashboard visualizes:

Revenue by region (bar chart)
Top products (horizontal bar chart)
Daily revenue trend (line chart)
Total revenue indicators (cards)
This provides a complete analytics workflow from raw data to visualization.
